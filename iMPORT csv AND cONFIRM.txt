1. Run the API
From src/server\Api (typical URL: http://localhost:5227).
2. Get an admin JWT
POST http://localhost:5227/api/auth/admin/login
Body:
{ "username": "YOUR_ADMIN_USERNAME", "password": "YOUR_ADMIN_PASSWORD" }
Copy the token value from the response.
3. Import the CSV (recommended)
From the repo root:
cd e:\Projects\ERP\scripts.\import-subjects-master-from-csv.ps1 `  -CsvPath "..\docs\subjects_master_import_template.csv" `  -BaseUrl "http://localhost:5227" `  -AdminToken "PASTE_TOKEN_HERE"
That calls POST /api/admissions/admin/class-xii-subjects, which replaces the whole catalog with the rows from your file. Success output includes importedRowCount.
4. Confirm
Open:
http://localhost:5227/api/admissions/class-xii-subjects?board=MBOSE&stream=ARTS
You should see items with id, subjectName, sortOrder.
